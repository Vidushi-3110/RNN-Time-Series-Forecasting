# RNN Time-Series Forecasting
Temperature prediction using RNN (Jena Climate Dataset)

## Objective
Predict future temperature using past climate data using RNN.

## Dataset
Jena Climate Dataset

## Steps
- Data preprocessing (date features + normalization)
- Sequence creation (144 time steps)
- RNN model with dropout
- Training with 20 & 50 epochs

## Evaluation
- MAE
- RMSE
- Graph comparison

## Result
Model trained with 50 epochs performed better.
